一、操作說明 

    打開build資料夾點選escape.exe即可開始遊戲

    (一) 主遊戲
        1.操縱方法 : 
            (1) 使用WASD控制前後左右移動、 滑鼠控制視角及身體轉向
            (2) 遊戲時鼠標隱藏,由畫面正中十字代替，Esc控制鼠標隱藏/顯現
            (3) 點擊物件可以觸發事件或遊戲
        2.遊戲目標 : 
            玩家需要透過點擊、觀察物件及通關線索遊戲來獲得線索，而後解開門上的問題，並通關最終遊戲，即可過關。

    (二) 線索遊戲-打地鼠
        1.操縱方法 :
            滑鼠快速點擊畫面中的雲朵(就像打地鼠)
        2.遊戲目標 :
            抓住足夠多的記憶碎片(雲朵)，才能想起線索

    (三) 線索遊戲-1A2B
        1.操縱方法 :
            輸入4位數字後按check，直到猜中為止
        2.遊戲目標 :
            猜到正確密碼解鎖檔案

    (四) 線索遊戲-連連看
        1.操縱方法 :
            選兩個配對，點擊的兩張圖片之間連一條最多可以彎折2次的線,其中線不可以穿過其他圖片，配對正確就會消除
        2.遊戲目標 :
            清除所有物品，看見最終線索

    (五) 通關遊戲-跑酷
        1.操縱方法 :
            (1) 使用wasd移動、ww連按跑步、qe左右側閃
            (2) 空白鍵跳躍，可二段跳
            (3) 設置sp系統，跑步、側閃、二段跳皆需消耗sp值
            (4) 滑鼠控制視角
        2.遊戲目標 :
            夠快地跑到終點，逃出去

二、故事背景說明 (含通關解答)

    (一) 遊戲背景 :
        某天主角和他父親在釣魚時，父親喝了飲料後被毒死了，主角嚇暈了過去，醒來後忘記很多事情，因此要在家裡找線索回想起來。
        最終發現部分冰塊裡有毒，並且負責帶冰塊的人是母親。

    (二) 通關解答 :
        1.出生後點眼前的魚竿是初始線索，需要通過遊戲-打地鼠才能得到，象徵抓住記憶碎片。
        2.出門後是客廳，未開燈前可以看到唯一盞燈對準桌上的空杯，代表它很重要。
        3.此外客廳裡的書本等也有提供線索，協助解開門上密碼
        4.點開杯子看見人物線索一，根據其線索在客廳找到指定瓶子
        5.點開瓶子看見對話劇情一，並且客廳的的解門線索才會出現，可以解開第一道門(寢室)(ans = 9)
        6.解開第一道門後2.的空杯裡多了飲料，再點一次看見人物線索二，並且寢室裡的解門線索才會出現，可以解第二道門(廚房)(ans = 19)
        7.並且寢室裡包含遊戲-1A2B，以及對話劇情二，都可以獲得線索
        8.解開第二道門後2.的空杯裡多了飲料冰塊，再點一次看見人物線索三，並且廚房裡的解門線索才會出現，可以解第三道門(最終解)(ans = 27 & 1)
        9.並且廚房裡包含遊戲-連連看，以及對話劇情三，都可以獲得線索
        10.解開第三道門後遊戲通關，進入通關遊戲跑酷，象徵逃跑

三、設計說明
    (一) 主遊戲(程式碼裡 game1~game4 資料夾以外的部分) : 
        1.各線索的顯現以及三個線索遊戲的顯現/隱藏是使用Canvas的隱藏/顯現來製作
        2.線索盡量設計串聯關係，解開一個才會出現下一個，增加遊戲性
        3.遊戲意義也盡量與故事本身有關連，避免突兀(見一、操作說明 之 各遊戲目標)
        4.當鼠標出現時停用畫面正中的十字以及固定畫面，鼠標隱藏時則停用鼠標功能(檢查Cursor.visible)，避免誤擊
        5.設定兩種背景音樂，平時為和平的，看線索時為驚悚的
        6.三種獲得故事線索方式 : 遊戲、圖片顯示、對話劇情
        7.各種合理的基本互動 : 開燈、開電視、酒醉、開火爐等
	8.場景部分，是選好純房子外殼後，下載各類家具進行擺設


    (二) 線索遊戲-打地鼠(程式碼裡的 game3) :
        1.設定雲朵浮現的速度使他可以變快
        2.淡入淡出的效果

    (三) 線索遊戲-1A2B(程式碼裡的 game1)

    (四) 線索遊戲-連連看(程式碼裡的 game2) :
        1.複雜的連線成功條件 (見一、操作說明 之 (二))
        2.具巧思的配對判定 :
            長方體-正方形
            圓柱體-長方形
            圓錐-圓形
            六角錐-三角形
            正方體-六邊形

    (五) 跑酷遊戲(程式碼裡的 game4) :
        1.採用sp系統，可以做出更多酷炫操作(見影片)
	2.具有衝刺效果
